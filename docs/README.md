# Documentation

Technical documentation for the current version of the ADHD-200 dynamic functional connectivity pipeline. It describes the implementation as it stands; past decisions and older versions are left out.

| Document | Description |
|----------|-------------|
| `architecture.md` | Modules and data flow. |
| `methodology.md` | How data are processed and experiments run. |
| `validation.md` | What has been verified in the implementation. |
| `performance.md` | Computational optimizations. |
| `limitations.md` | Scope and known limitations. |
| `auditoria-metricas.md` | Audit of the manuscript figures against the versioned results. |
| `modificaciones_eventanado.md` | Integration notes for the windowing changes. |

It complements the source code and the manuscript, and summarizes the relevant decisions without documenting every function.
